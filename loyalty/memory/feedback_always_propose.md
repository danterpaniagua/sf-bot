---
name: feedback-always-propose
description: Always include a concrete proposal or next step in every output
metadata:
  type: feedback
---

Never end a response by just presenting information and asking an open-ended "what do you want to do?" Always include a concrete proposal — recommended next action, suggested change, or specific option — and let the user redirect.

**Why:** User explicitly requested this. Passive outputs that only display data without a recommendation are not acceptable.

**How to apply:** Every response should close with at least one concrete proposed action, even if framed as "recommend X — redirect if needed."
